# Agro Survey

![Python](https://img.shields.io/badge/python-v3.7+-blue.svg)

## Sobre o Projeto
O Agro Survey é um projeto dedicado à coleta e análise de dados do mercado de trabalho de profissionais da área agrícola. Este repositório contém os códigos, datasets e documentação relacionados ao projeto.
